# Kinderschedule

Simple and effective weekly timecard calendar for preschool educators.

## Choose weeks to plan

Plan your workhours within a timeframe you set. The app does the counting
so you don't have to.

## Add educators

Add as many educators as you want. Print all timecards in one go for easy 
overview.

## Rules already in place

- 30 minutes drawn of from total time for breaks
- 1 hour drawn of on mondays if educator begins
working before 07:30.
